Необходимо создать новый Java проект в среде IntelliJ IDEA для разработки игры "Крестики-нолики".

Данная задача состоит из следующих подзадач:

Создать новый Java проект с именем tic-tac-toe

Создать новый локальный git репозиторий на базе созданного Java проекта

Установить commit-msg hook для созданного локального git репозитория, используя инструкции:
https://github.com/devonline-academy/tools/blob/master/git/commit-msg-verifier/Readme.md

Добавить файлы .gitignore, .gitattributes, Readme.md и файлы лицензии, которую Вы выбрали для Вашего проекта.
Для выполнения данного шага Вы можете использовать следующий шаблонный проект: https://github.com/devonline-academy/java-project-template

Сохранить изменения в систему контроля версий.

При необходимости: Задать имя пользователя и email, от имени которого будут сохраняться commits в системе контроля версий.
Данный шаг можно пропустить, если у Вас уже задан глобальный пользователь и Вы хотите сохранять commits от имени данного пользователя.

Сохранить изменения в систему контроля версий.

Создать удаленный репозиторий на сервисе github.com

Загрузить проект на удаленный репозиторий.

Необходимо создать корневой пакет и класс Launcher, который будет использоваться для запуска игры

При необходимости настроить IntelliJ IDEA для автогенерации шаблонного кода, согласно выбранной Вами лицензии (см. раздел Защита интеллектуальной собственности)

При необходимости настроить шаблон авторства в коде классов (см. раздел Защита интеллектуальной собственности)

Создать корневой пакет для проекта, например: academy.devonline.tictactoe

Создать класс Launcher, который будет использоваться для запуска созданного приложения.

В классе Launcher создать public static void main функцию с пустым телом. Тело данной функции будет создано позже в данном разделе!
